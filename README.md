# ThemeHub (Vite + React)

This is your company theme platform starter built with React on Vite.

## Features

- Company name input and slug generation
- Theme catalog with search
- "Open Theme" redirect with company slug as query param
- Admin form to add new themes (name, category, demo URL, code link)

## Run locally

```powershell
npm install
npm run dev
```

Then open the local URL shown in terminal (usually `http://localhost:5173`).
